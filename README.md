# skibidi-lang

A simple tree-walk interpreter for learning purposes.

## Features

#### Types
- `fp` - floating point numbers
- `int` - integers
- `str` - strings
- `bool` - booleans
- arrays

#### Operators
- `+`, `-`, `/`, `*`
- `==`, `!=`, `<`, `>`, `<=`, `>=`
- `AND`, `OR`, `NOT`

#### Variables
- mutable
- constant

#### Control Flow
- if statements
- while loops
  - break
  - continue

#### Functions
- function declarations
- return

#### Standard Library
- `print()`

#### Scoping
Basic scoping rules for variables and functions
